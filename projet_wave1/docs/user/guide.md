## 1. Introduction

**Wave** est une application web développée par l'équipe **Kobe** permettant d'explorer et d'analyser les contenus du magazine **Wired** à travers des visualisations interactives.

### Ce que vous pouvez faire avec Wave

- 📊 Explorer le **Co-occurrence Network** des mots-clés
- 📈 Consulter le **Radial Chart** des mots-clés par catégorie
- 🗓 Visualiser la **Heatmap** de distribution mensuelle des articles
- 🔍 Rechercher et filtrer les articles par catégorie et par date
- 💬 Nous contacter via le formulaire de contact

> ℹ️ Aucun compte utilisateur n'est nécessaire — l'application est accessible librement.

---

## 2. Accéder à l'application

Ouvrez votre navigateur et rendez-vous sur l'URL de l'application.

- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge

> 💡 **Astuce :** L'application est responsive et fonctionne sur smartphone et tablette.

## 3. Les visualisations

### 3.1 Co-occurrence Network

Le **Co-occurrence Network** est un graphe interactif qui relie les mots lorsqu'ils apparaissent ensemble dans un même titre d'article.

**Comment l'utiliser :**
- **Survolez un nœud** pour voir ses connexions avec les autres mots
- **Cliquez sur un nœud** pour charger les articles liés
- Plus un nœud est grand, plus le mot apparaît fréquemment

---

### 3.2 Radial Chart — Mots-clés par catégorie

Le **Radial Chart** affiche les mots-clés les plus fréquents sous forme de graphique radial. Chaque rayon représente un mot-clé, les couleurs indiquent les catégories.

**Comment l'utiliser :**
- Chaque **rayon** représente un mot-clé populaire
- Les **couleurs empilées** montrent les occurrences par catégorie
- Survolez une section pour voir le détail

---

### 3.3 Heatmap — Distribution mensuelle

La **Heatmap** affiche l'activité éditoriale du magazine Wired mois par mois. Les cellules plus foncées indiquent une activité plus élevée.

**Comment l'utiliser :**
- Les **lignes** représentent les années
- Les **colonnes** représentent les mois
- **Cliquez sur une cellule** pour charger les articles de cette période

## 4. Recherche et filtres

### Rechercher des articles

Utilisez la **barre de recherche** pour trouver des articles par mots-clés dans les titres.

### Filtres disponibles

| Filtre | Description |
|--------|-------------|
| **Catégorie** | Filtrer par thème (Tech, Science, Business...) |
| **Année** | Filtrer par année de publication |
| **Mois** | Filtrer par mois de publication |

## 6. FAQ

**Q : L'application nécessite-t-elle un compte ?**
R : Non, Wave est entièrement accessible sans inscription.

**Q : Les données sont-elles en temps réel ?**
R : Non, les données sont issues d'une base constituée à partir des archives du magazine Wired.

**Q : Sur quels navigateurs fonctionne Wave ?**
R : Chrome, Firefox, Safari et Edge dans leurs versions récentes.

**Q : L'application fonctionne-t-elle sur mobile ?**
R : Oui, Wave est responsive et s'adapte à tous les écrans.

**Q : Comment signaler un bug ?**
R : Via notre [GitLab Issues](https://scm.univ-tours.fr/22510981t/wave/-/issues/new) ou via le formulaire de contact.
